Programa Simples de Cálculo de IMC com HTML e JavaScript

Neste exemplo, o HTML fornecerá a estrutura para os campos de entrada de peso e altura, e um botão para iniciar o cálculo. O JavaScript será responsável por pegar esses valores, calcular o IMC e exibir o resultado na página.

Passo 1: Crie um arquivo HTML (por exemplo, imc.html):
Este arquivo conterá a estrutura da página com os campos de entrada e a área para exibir o resultado.

Explicação do Código:

  • HTML (imc.html): A estrutura básica da página define o título e estilos. O corpo contém inputs para peso e altura, um botão que aciona o cálculo e uma área de resultado.
  • JavaScript: A função calcularIMC() coleta os dados, verifica se são válidos, calcula o IMC e exibe o resultado formatado com duas casas decimais.

Como Usar:

  1. Salve o código como imc.html.
  2. Abra o arquivo no navegador.
  3. Digite peso e altura e clique no botão para ver o resultado do IMC.

Demonstração: Tudo acontece diretamente no navegador (frontend), sem necessidade de backend.
